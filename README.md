# Task Manager React App

A **Task Manager** web app built with **React**, **Redux Toolkit**, and **Material-UI**.  
Manage tasks with statuses, filters, and localStorage persistence. Fully responsive for desktop and mobile.

---

## Features

- Add tasks with text and category
- Toggle completion (COMPLETED / IN PROGRESS)
- Change status via dropdown (NEW / IN PROGRESS / COMPLETED)
- Delete tasks
- Filter by text and category
- LocalStorage persistence

---

## Tech Stack

- React (functional components & hooks)
- Redux Toolkit
- Material-UI (MUI)
- LocalStorage
